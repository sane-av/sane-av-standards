# SANE Standards

Community-voted technical standards for professional AV systems. Each standard is proposed, discussed, and ratified through the [RFC process](https://github.com/sane-av/sane-av.github.io/blob/main/CONTRIBUTING.md).

## Structure

Each standard is a single Markdown file in this repo's root. The filename is the standard's ID (e.g., `SANE-001-audio-levels.md`).
