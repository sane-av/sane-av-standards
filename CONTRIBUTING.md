# Contributing to SANE Standards

Standards follow the RFC process in the main SANE repository.

See [sane-av.github.io/rfc](https://sane-av.github.io/rfc) for the full process.

## Quick summary

1. Open an [RFC issue](https://github.com/sane-av/sane-av.github.io/issues/new?template=rfc.yml)
2. Community discusses and refines
3. Open a PR to this repo with the standards document
4. Maintainers review and merge on consensus
